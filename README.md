# pyqt5_android_tools
PyQt开发的协助安卓日常测试的小工具（截图，安装，卸载，logcat，anr，获取包名，获取设备信息）


开发环境配置：
https://www.jianshu.com/p/6fab1fe260d2

环境：
pip3 install PyQt5
adb 环境配置


执行：
python call_tools.py
